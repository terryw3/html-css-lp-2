Semantic HTML

Semantic Tags

Tag Description
<html> The root element of an HTML document.
<head> Holds metadata about the webpage without directly affecting the visibility.
<title> Title of the document, which appears in the browsers tab.
<meta> Provides metadata such as character encoding, viewport settings for responsive designs and other crucial details that assist in browser rendering and SEO.
<body> Contains the main content to be displayed on the webpage.
<header> Typically placed at the beginning of the webpage, this contains elements such as headings, logos or navigation menus.
<nav> Provides a structure for grouping and identifying navigation-related content within a webpage.
<main> Used for the primary content of a document, excluding headers, footers, and navigation elements.
<section> For grouping content within a document, helping to structure and organize related information in a way that enhances both the semantics and accessibility of the webpage.
<article> Used to define a self-contained, independent piece of content within a document, such as a news article, blog post, or forum post, allowing it to be distributed and reused independently while contributing to the overall document structure.
<aside> Used to represent content that is tangentially related to the content around it, often used for sidebars or content that can be considered separate but related.
<footer> Typically defines the footer of a section or a page, containing metadata, copyright information, or other related details.
